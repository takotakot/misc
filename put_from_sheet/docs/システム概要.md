# シート上のデータを PUT したい

## 概要

Google Sheets にデータを入力して、そのデータに応じて外部 API サーバにリクエストを行いたい。
本システムはその PoC をユーザーに示すものである。

## 入力画面

input シート

商品分類記入欄

列:

- `#`
- JAN
- 商品名
- 店舗に商品を置くフラグ
- 1店舗当たり売上数
- …

## 補助シート

filtered_items シート

「店舗に商品を置くフラグ」に「○」が入っている行のみを選択・表示するシート

## 動作概要

「店舗に商品を置くフラグ」に「○」が入っている行の JAN 列を抽出して、PUT リクエストを外部 API サーバに送信する

結果を表示して終了する

## 概要設計

- スプレッドシートにボタンを設置
   - 押すと関数が動く
- GAS で、データ取得してフィルタする部品と、PUT/POST する部品を作る
  - Google Sheets の QUERY 関数で、〇がついているのだけフィルタしたシートを別のひとつ追加する
- 必要であればログ出力シートあるいは結果画面（GAS は何故かシートに追加でモーダルウィンドウみたいなのが出せる）追加
